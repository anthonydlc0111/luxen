# luxen

Luxen es una landing page de generación de clientes potenciales para electricistas, esta landing page tendrá el potencial de convertirse en un sitio web o hasta incluso una aplicación movil.

Para llevar este proyecto acabo iré aprendiendo según necesite, pero principalmente me enfocare en estos lenguajes de desarrollo:

## Lenguajes y herramientas de desarrollo:

1. HTML.
2. CSS.
3. JavaScript.
4. Node.js (Backend):
   - Express.js.

5. MongoDB (Base de datos).
6. Integración de SMTP:
   - Nodemailer.

7. Deploy:
   - Vercel.
   - Render.

Por ahora esto es todo lo que iré aprendiendo y ejecutando en la marcha, los resultados se irán mostrando según valla avanzando.
